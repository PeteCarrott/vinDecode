# Vin Decode

## Link to project

<https://www.vindecodeapp.com/>

## Description

A web app to view detailed info about a vehicle from it's Vehicle Identification Number (VIN).

## Built With

* HTML & CSS
* Sass
* JavaScript
* Jest
* Webpack
* Babel
* Google Maps JavaScript API
* NHTSA API
* Netlify

Data comes from NHTSA
  
<https://vpic.nhtsa.dot.gov/>
